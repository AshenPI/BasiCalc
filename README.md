basicalc is a simple npm package that can do 
1- additon 
2- subtraction
3- multiplication
4- division
5- powers
<br>

package link : https://www.npmjs.com/package/basicalc


installation:
`npm i basicalc`

usage:

`
const basicalc = require("basicalc");

basicalc.add(3,2);
basicalc.subtrac(3,2);
basicalc.multi(3,2);
basicalc.divison(3,2);
basicalc.pow(3,2);
`
